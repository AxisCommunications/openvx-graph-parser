Class ImageAttributes

source code

Class that contains id and explicitly set attributes for an image node. Image attributes are specific properties set on images in the graph. Image attributes do not include image format and image type (e.g. input or debug image). These are currently handled separately. NOTE: When values for an attribute are referencing userdata attributes it is assumed that the reference is the first thing on the value and that there only is one reference per attribute. E.g. [width ref_width/2] is OK whereas [width 0.5*ref_width] is not.

Examples of how to specify image attributes in graph on an image node's data field: [width 100] [width ref_value*7] (Assuming ref_value is defined in userdata node)
